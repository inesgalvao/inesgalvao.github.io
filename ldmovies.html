LDMOVIES

Ld Movies was a project developed in group, in the context of the Computer Systems subject. The work aimed to create a website where it was possible to buy movies, add money, send money between users and leave comments. The work required the development of mockups and the respective implementation of the website (HTML/CSS/JavaScript) and of the database.
