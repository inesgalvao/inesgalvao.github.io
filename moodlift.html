MOOD LIFT

Mood Lift was a project developed in group, in the context of the chair of Interactive Media Design. The purpose of the work was the creation of an interactive prototype, of an application, whose theme was free. The only requirement was that it contributed or re-used content from the Europeana repository. MoodLift is a health and wellness application, which aims to help a user control their emotions and habits, through interesting activities. Every day, the user must answer a set of three questions. Through these answers the application can choose the activity to suggest the next morning.
